# Earthquake Challenge

The following challenge takes earthquake data and uses MapBox API to produce an interactive map with multiple layers set to different parameters.

## Tools
  - JavaScript
  - geoJSON
  - D3
  - MapBox/Leaflet
  - HTML & CSS
